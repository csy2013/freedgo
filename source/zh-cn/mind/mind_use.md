## 1、介绍

 Freedgo思维导图是一款简单好用的思维导图软件，除了可以轻松绘制不同风格的思维导图，支持各种不同的结构布局。支持导图文字、图形、备注、表格的样式设置，为客户提供美观漂亮的思维导图，可以把思维导图导出为图片格式，网页模式，通过提供打印。
 
 ![裕奥在线思维导图](https://www.freedgo.com/public/themes/freedgo/mind/mind_index.png "阿里云架构图")
 
Freedgo思维导图工具可以从如下格式文档导入:

- word文档(doc,docx)
- excel(xls,xlsx)
- pdf
- markdown
- html
- ppt文档包括(pptx)

![裕奥在线思维导图_导入文档](https://www.freedgo.com/public/themes/freedgo/mind/mind_import.png "裕奥在线思维导图_导入文档")    
    
    
Freedgo思维导图同时提供思维导图图片保存和文档保存方式,也可以导出如下文档:

- word文档(doc,docx)
- excel(xls,xlsx)
- pdf
- markdown
- html
- ppt文档包括(pptx)
![裕奥在线思维导图_导出文档](https://www.freedgo.com/public/themes/freedgo/mind/mind_export.png "裕奥在线思维导图_导出文档")
