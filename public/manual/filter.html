Filter

A filter is used to modify some specified data. Hexo passes data to filters in sequence and the filters then modify the data one after the other. This concept was borrowed from Online Diagram Drawing.

Synopsis

You can define the priority. Lower priority means that it will be executed first. The default priority is 10.

after_init

Executed when creating a post to determine the path of new posts.

post_permalink

Used to determine the permalink of posts.
